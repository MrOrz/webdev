屬性

通用屬性

HTML element 可以有屬性，用空白隔開。
大部分 HTML element 都有下面屬性：

  • class,id：替該 element 加上一個 classname / unique ID，使 CSS 的 selector 能選取 / 讓 javascript 方便操作。
  • title：工具提示（tooltip）
  • style：加上 CSS Declaration。勿和 <style> 標籤混淆。

屬性

  • src：圖片的相對路徑或絕對路徑
  • width, height：指定其一→等比例縮放
  • alt：替代文字
  • href：目標的相對路徑或絕對路徑
  • target：值為 "_blank" 時，會開新視窗

Nesting Rule

  • 原則上 inline element 不能包 block element
  • <p>、<h1>~<h6> 裡頭只能包 inline element
  • <ul>、<ol> 裡面只能包 <li>
  • <tr> 裡面只能包 <td> 或 <th>，外頭必須要有 <table>

HTML 文件架構

  • <head> 存放「不是網頁內文」的其他資訊。
  • <title> 會顯示在瀏覽器分頁標題的主題。
  • <meta> 放關於網頁的其他資訊，有 name 和 content 屬性。
  • <body> 放網頁內容。
  • <html> 包住 <head> 和 <body>。
