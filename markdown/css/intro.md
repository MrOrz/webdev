長這樣
-----

```css
html, body {
  position: relative;
  padding: 0; margin: 0;
  overflow: hidden;
  font-family: '文泉驛微米黑', 'LiHei Pro', '微軟正黑體', sans-serif;
  font-weight: 200;
  letter-spacing: -0.02em;
  color: #eee; width: 100%; height: 100%;
  min-height: 400px;
}

table {
  border-collapse: collapse;
  border-spacing: 0;
}
```

把 CSS 加進 HTML
---------------

### `<link>`

```html
<head>
  ...
  <link rel="stylesheet" type="text/css" href="path/to/file.css">
  ...
</head>
```
